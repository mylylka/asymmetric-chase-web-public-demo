# Asymmetric Chase Public Demo

## Purpose
This folder is a lightweight public web package for the asymmetric chase prototype.

## Structure
- `index.html`: public entry for the latest playable build.
- `css/`, `js/`: runtime files used by the public build.
- `assets/devices/`: optimized device art.
- `assets/游戏图样/角色/optimized/`: optimized character sprites only.
- `demos/`: archived standalone demo pages for review and storytelling.

## Rules
- Keep this package public-safe. Do not add secrets, private notes, raw source images, or local cache files.
- Do not copy large original artwork here unless it is actually loaded by the public game.
- Prefer optimized images and static files that can run on GitHub Pages.
- Keep the full development history in the private source repository.

## Verification
- Open `index.html` through a local static server before publishing.
- Confirm the latest game loads and the demo links are present.
